# Coup

A CLI implementation of the card game Coup — 1 human vs 2–3 AI opponents.

## About

Full rules implementation with bluffing, challenges, and blocking. Features four AI personality presets:

- **Gambler** — aggressive, takes risks
- **Accountant** — conservative, plays the odds
- **Grudge** — targets whoever wronged them last
- **Ghost** — unpredictable, hard to read

Supports multi-round match mode.

## Usage

```bash
python3 coup.py
```

Requires Python 3.8+. No dependencies.
